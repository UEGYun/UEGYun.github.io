# 내 맘대로 해석 하기
자훈: 하나의 내용을 영어, 한자, 한글명, 한글뜻 따로 몇번씩 외우지 말자. (영어명,한글뜻 두개만 알면 되지)
<br>Class(클래스) == 틀
<br>Struct(스트럭트) == 고정틀
<br>Instance(인스턴스) == 만든것
<br>parameter(파라메터) == 입력값
<br>variable(바) == 변수
<br>const(컨스트) == 고정화
<br>static(스태틱) == 고유화
<br>Runtime(런타임) == 프로그램 운영 소프트웨어
